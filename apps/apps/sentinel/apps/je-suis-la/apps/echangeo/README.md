# ÉCHANGEo

Plateforme d’entraide locale de l’écosystème Sentinel Global.

Objectif :
Permettre aux personnes proches géographiquement de s’entraider simplement
en échange de services, compétences ou temps.

ÉCHANGEo fait partie de l’écosystème :

- Sentinel (sécurité)
- Je suis là (signal de présence)
- ÉCHANGEo (entraide locale)

Toutes les applications partagent :

- un design commun
- une authentification commune
- une architecture technique commune
